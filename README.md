# Blue Chip Staffing LLC Website

Modern, professional staffing agency website built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- Responsive design for all devices
- 22 industry landing pages (including 7 new ones)
- Team/leadership page with org chart
- Animated scroll reveals and transitions
- Contact form (Formspree integration ready)
- SEO optimized
- Modern blue color palette matching brand

## Pages

- `/` - Homepage
- `/about` - About Us with team bios
- `/employers` - Employer services
- `/job-seekers` - Job seeker resources
- `/contact` - Contact form and locations
- `/industries` - All industries overview
- `/industries/[slug]` - 22 individual industry pages

## Quick Start

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
vercel
```
